# gym-pwa

Для сессионного подхода (одна тренировка = одна запись с началом/концом) выполни в Supabase SQL из файла `supabase-workout-sessions.sql` — создаётся таблица `workout_sessions`. После этого на главной появятся «Начать тренировку», таймер «Идёт тренировка» и кнопка «Завершить» с экраном итогов.
